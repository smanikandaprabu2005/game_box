 Game Box
 
Game Box is a browser-based game hub that brings together multiple classic games in one interactive platform.  
Built using HTML, CSS, and JavaScript, it features a visually appealing 3D slider interface and works seamlessly on GitHub Pages.

 Games Available
-  Ludo  
-  Memory Card Game  
-  Ping Pong  
-  Snake and Ladder  

Each game opens in a new tab and runs independently.

 Key Features
 
- Interactive 3D carousel UI
- Pure frontend project (no backend)
- GitHub Pages compatible
- Responsive layout
- Clean and organized file structure
- Beginner-friendly JavaScript logic

Tech Stack

- HTML5
- CSS (Animations & Transforms)
- JavaScript
- Git & GitHub
- GitHub Pages (Deployment)
